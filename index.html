Gene Browsepad

Ever wondered what would happen if you combine an application for accessing information and a text-editor to create your documents?
We bring you Gene, the Browser+Notepad application which is an open source initiative under GPL license.
It is a basic application made with Python and PyQt GUI widget toolkit. It helps web designers to see the output as and when they edit their local .html, .css and .js files when they press Ctrl+s. It also highlights the basic syntax to help the web designers improve the readability and context of the text.
